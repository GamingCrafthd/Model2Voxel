# Model2Voxel
> Calculates the amount of cm from model gauges to voxels.

## Formula
`v = i * g / 100 * 16`<br>
`v` is result.<br>
`i` is the length in cm.<br>
`g` is the gauge. (For example: g = 87 (Gauge H0))

## Usage example

Enter the length in cm.
Click on calculate.
Result will appear next to the calculate button.
